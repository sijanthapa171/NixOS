# NixOS Installation Guide

This guide will walk you through installing our NixOS configuration system, which provides a powerful, modular, and reproducible setup designed for modern computing needs.

## Prerequisites

Before beginning the installation, ensure you have:

- A NixOS installation or live environment
- Internet connection
- Basic understanding of Nix/NixOS concepts
- At least 50GB of free disk space
- 8GB RAM (16GB recommended)

<Note title="Important" type="warning">
  Make sure to back up any important data before proceeding with the installation.
  While the process is designed to be safe, it's always good practice to have backups.
</Note>

## Installation Methods

<Stepper>
<StepperItem title="1. Clone the Repository">
First, clone the NixOS configuration repository:

```bash
git clone https://github.com/authxt/NixOS.git
cd NixOS
```
</StepperItem>

<StepperItem title="2. Review Configuration">
Before proceeding, review and adjust the configuration in `flake.nix` according to your preferences:

```nix
settings = {
  username = "your-username";
  editor = "nixvim";    # Options: nixvim, vscode, nvchad, neovim
  browser = "floorp";   # Options: firefox, floorp, zen
  terminal = "kitty";   # Options: kitty, alacritty, wezterm
  videoDriver = "nvidia"; # Options: nvidia, amdgpu, intel
};
```
</StepperItem>

<StepperItem title="3. Choose Installation Method">
There are two installation methods available:

**For existing NixOS systems:**
```bash
./install.sh
```

**For fresh installations:**
```bash
./live-install.sh
```
</StepperItem>

<StepperItem title="4. Post-Installation">
After the installation completes, rebuild and switch to the new configuration:

```bash
nixos-rebuild switch --flake .#Default
```
</StepperItem>
</Stepper>

## Available Configurations

The system comes with several pre-configured options that you can choose from:

| Category      | Options                                | Description                               |
| ------------- | -------------------------------------- | ----------------------------------------- |
| Editors       | nixvim, vscode, nvchad                 | Pre-configured editor setups              |
| Browsers      | firefox, floorp, zen                   | Browser configurations with privacy focus |
| Terminals     | kitty, alacritty, wezterm              | Modern terminal emulators                 |
| File Managers | yazi, lf                               | Terminal and GUI file managers            |
| Themes        | astronaut, black_hole, purple_leaves   | SDDM and system themes                    |

## Project Structure

The configuration is organized in a modular structure:

```
NixOS/
├── modules/           # Core system modules
│   ├── desktop/      # Desktop environments
│   ├── hardware/     # Hardware configurations
│   ├── programs/     # Application setups
│   └── themes/       # Visual customizations
├── hosts/            # Host-specific configs
├── pkgs/            # Custom packages
└── overlays/        # Nixpkgs overlays
```

## Troubleshooting

If you encounter any issues during installation:

1. Check the system logs for error messages
2. Ensure all prerequisites are met
3. Verify your hardware configuration matches the settings
4. Visit our [GitHub repository](https://github.com/authxt/NixOS) for additional help
